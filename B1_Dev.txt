B2---------------------------------------------------------------


1. Tag Your Image

Docker requires the image to include your Docker Hub username before pushing.

docker tag imgname your_dockerhub_username/imgname:latest


ex : docker tag name1 likhithgm/name1:latest

2. Push the Image
docker push your_dockerhub_username/imgname:latest


ex: docker push likhithgm/name1:latest



⬇️ To Pull the Image Later
docker pull your_dockerhub_username/imgname:latest
